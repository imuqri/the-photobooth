## [1.0.0] — initial scaffold
### Added
- Signaling server: room create/join, WebRTC offer/answer/ICE relay, position sync, room locking, rate limiting, in-memory TTL rooms.
- Frontend: landing page (create/join, no accounts), room page with live WebRTC mesh, MediaPipe background segmentation, canvas compositor with drag-to-position, 3-photo strip and 4-photo grid capture flow, direct browser download.
- CI workflow for build checks on PR/staging/main.
